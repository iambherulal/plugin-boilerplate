Plugin Boilerplate Generator is a powerful tool to streamline your WordPress Plugin Boilerplate creation process.

== Description ==

Plugin Boilerplate Generator simplifies creating and installing customized WordPress Plugin Boilerplate. It's allowing you to generate code tailored to your project's needs.
You can generate a plugin boilerplate with a single click and install it directly into your WordPress site.

== Features ==

- **Customizable Boilerplate Code:** Tailor the generated code to match your project's needs.
- **One-Click Installation:** Install the generated plugin directly from the tool for a seamless workflow.
- **Time-Saving and Efficient:** Save time and boost efficiency in your plugin development process.
- **Tailored to Your Preferences:** Define your preferences, and let the generator create code that aligns with your development style.

== Installation and Use ==

1. Upload the entire `plugin-boilerplate-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the Tools menu in the WordPress admin and click on "Plugin Boilerplate"
